# ansible-role-ctop

Ansible role that installs [ctop](https://github.com/bcicen/ctop).

# Requirements

None

## Role Variables

All variables are listed below (see also `defaults/main.yml`).

```yml
ctop_version: "latest"
```

# Usage

Include the role as usual:

```yml
ansible.builtin.import_role: frozenfoxx.ctop
```
